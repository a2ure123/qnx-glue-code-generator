atol(), atoll()

Updated: October 28, 2024

Convert a string into a long integer

Synopsis:

#include <stdlib.h>

long atol( const char* ptr );

long long atoll( const char* ptr );

Arguments:

ptr
A pointer to the string to parse.

Library:

libc

Use the -l c option to qcc to link against this library. This library is usually included automatically.

Description:

The atol() function converts the string pointed to by ptr to a long integer; atoll() converts the string pointed to by nptr to a long long integer.

Returns:

atol()
A long integer.
atoll()
A long long integer.

Examples:

#include <stdlib.h>
#include <stdio.h>

int main( void )
{
    long x;

    x = atol( "-289" );
    printf( "x = %d\n", x );
    return EXIT_SUCCESS;
}

produces the output: 

x = -289

ANSI, POSIX 1003.1
